# Figure 3
Senescence- and plasticity-associated transcriptional programs across pediatric lung development and alveolar deficiency (BPD).
Panels A–F summarize overlaps of differentially expressed genes identified in epithelial, stromal, and transitional cell populations across developmental stages and BPD.
Gene lists were generated from differential expression analyses performed on the integrated Seurat object. Venn diagrams were generated using the Venny online tool (https://bioinfogp.cnb.csic.es/tools/venny/).
Final figure assembly was performed using PowerPoint.
