This folder contains developmental expression analyses of epithelial marker genes across pediatric lung development and PT-BPD samples.

---

## Supplementary Figure 3A – SFTPC Expression

FeaturePlot visualization of SFTPC expression across developmental stages and PT-BPD samples.

### Workflow

- Cells were grouped by developmental stage.
- Normalized expression values were extracted from the integrated Seurat object.
- FeaturePlots were generated to visualize SFTPC expression across the UMAP embedding.

### Software

- R
- Seurat
- ggplot2
